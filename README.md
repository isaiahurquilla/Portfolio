# Portfolio Website

A personal portfolio website built to showcase projects, blog content, resume information, and contact details in one place.

## Tech Stack
- React
- JavaScript (JSX)
- React Router
- CSS

## Deployment
This portfolio is a frontend-only React application, so it can be deployed as a static site on platforms like **Vercel**.

For project demos:
- **DBCore4U Clothing Storefront** has a public live demo hosted on Vercel.
- The other two mobile projects currently run **locally through Expo Go**, so they do not have public live demo links yet.


## Known Limitations
- Two mobile projects do not currently have public hosted demos.
- Some project details are frontend-only and do not include backend integrations or a CMS.
