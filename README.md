# Portfolio Website

A personal portfolio website built to showcase my projects, blog posts, resume, and contact information in one place.

## Tech Stack
- React
- Vite
- JavaScript (JSX)
- React Router
- CSS

## Deployment
This portfolio is a frontend-only React application built with Vite, so it can be deployed as a static site. The site is deployed using Vercel, which connects to the GitHub repository and automatically rebuilds the app when changes are pushed.

For project demos:
- **DBCore4U Clothing Storefront** has a live demo hosted on Vercel.
- **About Me App** and **Nutrition Tracker** are mobile projects and currently run locally through Expo Go, so they do not have public live demo links.

## Known Limitations
- Two featured projects are mobile apps and do not currently have public hosted demos.
- The portfolio itself is frontend-only and does not include a backend or CMS.
- Project content is manually maintained rather than managed through an admin dashboard.
